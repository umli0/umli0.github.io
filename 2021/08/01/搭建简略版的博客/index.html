搭建博客教程

发表于 2021-08-01 更新于 2021-08-02

搭建博客教程

1.创建github账户，创建github仓库

2.建立连接，本地与github

2.1安装git

下载，安装git之后

2.2本地创建密钥

创建密钥

最后这个密钥会保存在某个文件下!
c盘/用户/用户名/.ssh/id_rsa下

2.3粘贴该密钥至github

库---setting---ssh与GPG keys下，输入密钥名称（自己造一个）和之前那个密钥文件下的所有内容
注是加那个.pub的
然后git bash下，ssh -T git@github.com

3. 先去官网下载安装包，安装node.js，再安装npm

然后用命令行

4.安装hexo 和本地预览博客

创建一个文件夹作为博客的目录，右键 git bash

npm install -g hexo-cli //安装hexo

然后可以打开浏览器
localhost:4000 //在本地上预览博客

ctrl+c可退出

网页预览博客

根目录下的_config.yml是站点配置文件
themes中的_config.yml是主题配置文件

为了让hexo和github关联起来，站点配置文件
ctrl+f 搜索deploy

配置

deploy:
type: git
repo: 仓库的路径 （是咋写的忘了）git@github.com:umli0/umli0.github.io.git
有个说要写成上面那样
很多就是直接仓库地址
branch: master

然后保存
然后安装插件

npm install hexo-deployer-git --save

hexo clean 

hexo g 

hexo d  //部署网站（这一步需要输之前ssh的密码）
也可以直接写成
hexo clean && hexo g && hexo d

然后就可以打开网站了xxx.github.io

更换主题

克隆主题至目录下themes下的next文件夹下
然后是配置主题文件

看别人写的吧 配置方法.

博客的本地目录下---git bash--用命令 git clone https://github.com/theme-next/hexo-theme-next themes/next   将想要下载的主题下载下来（这个是旋律NexT主题）
先下载好主题再换，如果先更改了主题，你会发现好像主题下不下来
更改站点的
theme: landscape
改为你新下的主题,theme下是啥，你就写啥

连接的时候报错了
fatal: unable to access ‘https://github.com/theme-next/hexo-theme-next/': Failed to connect to github.com port 443: Timed out

是说超时了吗，可以再重新输入一次命令

成功后

进入站点配置文件，配置theme: next

回到根目录 git bash 
hexo s --debug
然后本地预览查看是否无误

然后输入生成和发布命令，在网站上预览结果
好像是空白
但是按ctrl+f5多几次就可以运行了
ok完结
