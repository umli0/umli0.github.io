修改原理
主题配置文件_config.yml 中有一个关键词 custom_file_path，在这里可以设置自定义的的样式文件的路径。利用自定义的样式文件覆盖掉默认样式。

如

更换背景

1. hexo的根目录下\source\_data\styles.styl

如果没有就自己创建一个（不是next主题的根目录）

2.next目录下_config.yml取消注释

custom_file_path:
  #head: source/_data/head.swig
  #header: source/_data/header.swig
  #sidebar: source/_data/sidebar.swig
  #postMeta: source/_data/post-meta.swig
  #postBodyEnd: source/_data/post-body-end.swig
  #footer: source/_data/footer.swig
  #bodyEnd: source/_data/body-end.swig
  #variable: source/_data/variables.styl
  #mixin: source/_data/mixins.styl
  style: source/_data/styles.styl

style那取消注释

3.然后在styles.styl的文件中添加

body {
  background-image: url(https://c-ssl.duitang.com/uploads/item/201509/12/20150912214813_ChWRn.jpeg);
  background-repeat: no-repeat;
  background-attachment: fixed;
  background-position: 50% 50%;
  background-size:cover;
}

网页上能找到很多一大串配置代码，但是不建议一次性全部粘贴上去，不然出现错误也不知道在哪，页面的css样式都显示不出来

改变透明度

参考了shijy16的博文

1.

themes\next\source\css\_schemes\Pisces\_layout.styl
中

.content-wrap {
  background: rgba(255,255,255,1);//完全不透明

控制透明度

.content-wrap {
  background: rgba(255,255,255,0.5);//将透明度设为0.5

若是改成 background: white;就成完全不透明了

2.styles

根目录下source_data\styles.styl

增加

.main-inner { 
    margin-top: 60px;
    padding: 60px 60px 60px 60px;
    background: rgba(255,255,255,0.9);
    min-height: 500px;
}

//侧边框的透明度设置
.sidebar {
  opacity: 0.9;
}

//菜单栏的透明度设置
.header-inner {
  background: rgba(255,255,255,0.9);
}

//搜索框（local-search）的透明度设置
.popup {
  opacity: 0.9;
}

.footer-inner {
  background: rgba(255,255,255,0.8);
}

更改网站图标

themes\next\source\images

存图标地址

主题配置_config.yml下的favicon

favicon:
  small: /images/图标名16*16.png
  medium: /images/图标名32*32.png
  apple_touch_icon: /images/apple-touch-icon-next.png
  safari_pinned_tab: /images/logo.svg

可以自己去图标网找图标下载，也可以用图片自己做，自己调整大小（用电脑自带的图片工具-调整大小 即可）

修改字体

主题配置文件下

font:
  enable: ture
  host:
  global:
    external: true
    family: Noto Serif SC //配置这里，这里选择想要的字体
    size:

去这里找字体不用下载，记个名字就行。
