VUE 组件化思想

组件化

(1)传统的页面，会将大量的HTML/CSS/JS进行引入，但是引入之后结构混乱，不便于管理，开发维护成本比较好
(2)在VUE中，可以将一个组件，看作是一个页面，在其中可以引入独立的样式 JS/HTML进行单独管理
(3)组件也可以复用

组件入门案例

(1)组件入门写法
(2)局部组件

VUE中的理由

路由

(1)用户发起一个请求，在互联网中经过多个站点的跳转，最终获取服务器端的数据，把互联网中网络的链路称之为路由（网络用语）
(2)VUE中的路由，根据用户的请求URL，展现特定的组件（页面）信息，（控制用户程序跳转的过程）

路由步骤

(1)导入路由的JS文件
(2)指定路由跳转链接
(3)定义路由的填充位
(4)封装组件信息，指定路由对象
(5)在路由中声明路由

安装node.js

1、双击安装软件
2、控制台输入 node -v 查看版本号
3、控制台输入 npm -v 查看版本号
4、切换淘宝NPM库
(1)npm config set registry http://registry.npm.taobao.org
(2)npm config list

安装vue客户端

(1)npm install -g @vue/cli –force
(2)安装完之后，检测版本号 vue -V
6、运行vue客户端
(1)vue ui
