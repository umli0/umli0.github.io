el表达式

一套简单的计算规则，用于给jsp标签的属性赋值，也可以直接输出

使用el表达式

访问 entity属性的方法

  • ${user.name}

1.${user.name}:容器会依次从pageContext，request，session，application中查找绑定名为“user”的对象，然后调用对象的getName方法，输出name值

2.如果依据绑定明找不到对应的对象，会输出空字符串

3.对于null，会转化成空字符串输出，如果要制定查找范围，可以使用

  1. pageScope
  2. requestScope
  3. sessionScope
  4. applicationScope

代码

  • $(user[“name”])

1.这种方式允许[]里面出现变量
2.允许[]里面出现从0开式的下标用来访问数组中的某个元素
