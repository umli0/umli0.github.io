forEach标签（循环标签）

用来遍历一个集合或者数组
<c:forEach var=”” items=”” varStatus=””>
Items:用来指定要遍历的集合或者数组，可以使用el表达式（从request上获取绑定数据）
var：属性制定一个绑定名称
绑定范围是pageContext，容器每一次从集合或者数组中去一个元素，然后该元素绑定到pageContext对象上（每一次循环取出的元素给它命名）
等同于：for（int i=0;i<users.size();i++）{
User user = users.get(i)
}
varStatus：属性：指定一个绑定名称
绑定范围是pageContext，绑定值是一个容器创建的特殊对象，该对象封装了当前遍历的状态，比如搞对象提供了 getIndex()：当前正在被遍历的元素的下标（从0开始）

AJAX （asynchronous javascript and xml）

1、ajax定义
(1) Ajax是一种用来改善客服体验的技术，实质是利用浏览器提供一个特殊对象（XMLHttpRequest）异步地想服务器发送请求，服务器一般只需要返回部分数据（文本或者xml文档），浏览器利用这些数据更新当前页面，这个过程，页面局部刷新，不打断用户的操作

异步

异步：整个页面刷新——》同步，异步就是ajax对象向服务器发送请求时，浏览器不会销毁当前页面，用户仍可以对当前页面做其他操作

ajax对象的几个重要的属性

1）onreadystatechange：用来绑定一个事件处理函数（该函数用来处理readystatechange事件）
a. 当ajax对象的readyState属性发生了改变，比如从0变成了1，就会产生readystatechange事件
b. readState：有5个值，分为时0，1，2，3，4 表示ajax对象已经获得了服务器返回的所有的数据
c. responseText：获得了服务器返回的文本
d. responseXML：获得服务器返回的xml文档
e. status：获得服务器返回的状态码

④编程步骤
1）获取ajax对象
a. 比如 var = getXhr();
2) 使用ajax对象发送请求
a. get请求
a) xhr.open(‘get’,’check_username?uname=’+uname.value,true);
b) xhr.onreadystatechange=f1
c) xhr.send(null)
d) true:发送异步请求（浏览器不会销毁当前页面，用户仍然可以对当前页面做其他操作）
e) False：发送同步请不求(浏览器不会销毁当前页面，浏览器会锁定当前页面，用户不能够对当前页面做其他操作)
2）编写服务器程序，一般不需要返回完整的也面，只需要返回部分数据
3）编写事件处理函数（回调函数

jQuery
（1）jQuery介绍
① 是一个js框架（jquery.js）,特点就是利用选择器查找要操作的节点（dom对象，html各个标签），然后将找到的借点封装成jQuery对象，通过调用JQuery对象的方法或者属性来实现对底层的被封装的节点的操作，这样做的好处就是，代码有更好的兼容性从，也会更加简洁
② JQuery对象和dom对象之间如何转换

  1. Dom–>JQuery
    a. $(dom对象)
  2. JQuery–>Dom
    a. $obj.get(0)
    b. $obj.get()[0]
    ③选择器 select
    1）基本选择器
    a. #id
    b. . Class
    c. Element
    d. 多选择器
    2）层次选择器
    select1 select2 考虑所有后代
    select1>select2 只考虑子节点
    select1+select2 下一个兄弟节点
    select1~select2 下面所有兄弟节点
    3）过滤选择器
    a.基本过滤选择器
    :first
    :last
    :not(select)
    :even
    :odd
    :eq(index)
    :gt(index)
    :lt(index)
    b.内容过滤选择器
    a) :contains(text) 匹配包含给定文本的元素
    b) :empty 匹配所有不包含志愿书或者文本的空元素
    c) :has(select) 匹配包含有选择器所匹配的元素的元素
    d) :parent 匹配含有元素或者文本的元素
    c. 可见性过滤选择器
    a) :hidden 匹配所有不可见元素或者type为hidden的元素
    b) :visible匹配所有的课件元素
    d. 属性过滤选择器
    e. 子元素过滤选择器
    a) nth-child(下标从1开始)
