#（1）jQuery介绍
js简介
JQuery是一个js框架（jquery.js）,特点就是利用选择器查找要操作的节点（dom对象，html各个标签），然后将找到的节点封装成jQuery对象，通过调用JQuery对象的方法或者属性来实现对底层的被封装的节点的操作，这样做的好处就是，代码有更好的兼容性从，也会更加简洁
