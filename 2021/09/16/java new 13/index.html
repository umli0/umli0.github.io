发表于 2021-09-16 更新于 2021-09-23 分类于 java进阶课程
VUE第三方开发框架（上）

VUE

前端的框架

1.第三方开发的——下载
2.基于MVVM设计模式，（编程语言中设计模式，单例模式，工厂模式，适配器模式等等…）
3.设计模式：代码编程的规范
4.JAVAweb阶段 通用的设计模式 MVC
M modek 模型层 dao (设置具体的功能)
V view 视图层 前端页面 （设置前端的页面显示）
C control 控制层 servlet（设置某处调用某dao的功能）
5.渐进式的——可以逐步在项目中使用vue的部分继承，而不要求必须吧整个项目都改成vue
6.纯前端js——只要有浏览器就可以运行，和后端无关
7.框架——自动化，避免大量重复劳动

用框架的原因

简化开发，避免重复劳动
之后凡是以数据操作（增删改查）为主的项目，都适合用vue等框架
问题：兼容性问题，用着不习惯

如何使用框架

下载安装

下载：cn.vuejs.org1.

开发版本：未压缩版，包含完整的注释，代码格式和见名知意的变量名，而且还包含及其有好的错误提示信息，适合学习和开发使用、
2.生产版本：压缩版，去掉了所有的注释，代码格式，极简化了变量名，而且去掉了友好的错误提示信息，适合生产环境上线使用

MVVM设计思想

事件绑定

V-on:click
v-once：

双向的数据绑定

(1) V-model:实现页面也数据的绑定，页面边，数据变；数据变，页面也变
(2) 按键触发机制 v-on:keydown/keyup/keypress
支持的按键：回车，Tab，delete，ESC，空格，上下左右

按键修饰符：

(1) 阻止事件冒泡 .stop
<button @click.stop="num++">计算</button>阻止@click后发生的事

(2) 阻止默认行为 .prevent
a标签默认行为是href跳转页面，用户点击a标签，不跳转页面，同时触发事件
<a href="http://baidu.com" @click.prevent="aClick">百度</a>

声明变量的方式

    1.var 特点：没有作用域 
    2.let 特点：作用域var类似，有作用域的概念
    3.const :定义常量
